Documentation

Welcome to the Voca documentation. Voca is a powerful, non-interactive network downloader designed for speed, reliability, and ease of use.

Select a topic from the sidebar to explore features, protocols, and configuration options.

Quick Start

Install Voca using the command line:

curl -fsSL voca.riteshrana.engineer/install.sh | sh
Note: This documentation is currently being expanded. Detailed guides for specific protocols (HTTP, HTTPS, FTP) and features are being added regularly.
