# Voca Releases

Official binary releases for [Voca](https://voca.riteshrana.engineer) — a powerful, non-interactive network downloader.

## Quick Install

```bash
curl -fsSL voca.riteshrana.engineer/install.sh | sh
```

## Manual Download

Visit [voca.riteshrana.engineer](https://voca.riteshrana.engineer) or check the [Releases](https://github.com/ambicuity/voca-releases/releases) page.

### Available Platforms

| Platform | Architecture | File |
|----------|-------------|------|
| Linux | x86_64 | `voca-<version>-linux-x86_64.tar.gz` |
| Linux | ARM64 | `voca-<version>-linux-arm64.tar.gz` |
| macOS | x86_64 (Intel) | `voca-<version>-macos-x86_64.tar.gz` |
| macOS | ARM64 (Apple Silicon) | `voca-<version>-macos-arm64.tar.gz` |
| Windows | x64 | `voca-<version>-windows-x64.zip` |

## Verify Downloads

Each release includes `SHA256SUMS.txt`. Verify your download:

```bash
sha256sum -c SHA256SUMS.txt
```
